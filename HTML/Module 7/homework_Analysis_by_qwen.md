### Подробный разбор ошибок:

#### 1. Критическая ошибка: Перепутаны `figure` и `figcaption`
*   **Как у вас:**
    ```html
    <figcaption>
        <picture>...</picture>
        <figure>Подпись</figure>
    </figcaption>
    ```
*   **Правило:**
    *   `<figure>` — это **контейнер** (обертка) для медиа-контента (картинки, схемы, кода) и его подписи.
    *   `<figcaption>` — это **подпись** к этому контенту. Она должна лежать **ВНУТРИ** `<figure>`.
*   **Ошибка:** Вы сделали наоборот: обернули всё в подпись (`figcaption`), а саму подпись назвали контейнером (`figure`). Это полностью ломает смысл.
*   **Исправление:**
    ```html
    <figure>
        <picture>
            <source ...>
            <img ...>
        </picture>
        <figcaption>Вид на вершину Эльбруса</figcaption>
    </figure>
    ```

#### 2. Ошибка вложенности: Списки внутри `ol`
*   **Как у вас:**
    ```html
    <ol>
        <li>День 1...</li>
        <ol> <!-- Вложенный список -->
            <li>Подъем...</li>
        </ol>
        <li>День 2...</li>
    </ol>
    ```
*   **Правило:** Элементы списка (`<li>`) могут содержать внутри себя другие списки. Но сам тег списка (`<ol>` или `<ul>`) не может быть прямым потомком другого списка (`<ol>`). Он должен лежать внутри `<li>`.
*   **Ошибка:** У вас `<ol>` лежит прямо в `<ol>`, минуя `<li>`.
*   **Исправление:**
    ```html
    <ol>
        <li>
            День 1: Заброска...
            <ol> <!-- Вложенный список внутри пункта li -->
                <li>Подъем...</li>
                <li>Плавный набор...</li>
            </ol>
        </li>
        <li>День 2...</li>
    </ol>
    ```

#### 3. Ошибка дублирования `figcaption` вне `figure`
*   **Как у вас:**
    ```html
    <figcaption>
        <img src="...">
        <figure>Фото привала...</figure>
    </figcaption>
    ```
*   **Ошибка:** Та же самая, что и в пункте 1. Плюс, тег `<figcaption>` не может существовать сам по себе, он всегда должен быть привязан к родительскому `<figure>`.

#### 4. Аватарка автора
*   **Как у вас:** Аналогичная ошибка с `figcaption` и `figure`.
*   **Исправление:**
    ```html
    <figure>
        <img src="..." alt="...">
        <figcaption>Иван Иванов, гид</figcaption>
    </figure>
    ```

#### 5. `alt` атрибуты
*   **Как у вас:** `alt="Вид на вершину Эльбруса(для декстопа).jpg"` и подобные.
*   **Комментарий:** В атрибуте `alt` должно быть **описание картинки**, а не имя файла или техническая пометка.
*   **Исправление:** `alt="Панорамный вид на заснеженную вершину Эльбруса"`.
