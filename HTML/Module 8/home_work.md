### Практическое задание: «Страница музыкального альбома и клипа»

Вам необходимо сверстать страницу презентации нового музыкального альбома исполнителя. Страница должна содержать следующие элементы:

1.  **Шапка сайта:**
    *   Название лейбла или имя артиста.
    *   Меню навигации из трех пунктов (список).

2.  **Основной контент (Презентация):**
    *   **Заголовок страницы:** Название альбома (самый крупный заголовок).
    *   **Обложка альбома (Адаптивная):**
        *   Изображение должно меняться в зависимости от ширины экрана (мобильная версия — квадратная обложка, десктоп — широкая панорама с артистом).
        *   Используйте соответствующие теги для адаптивности.
        *   Изображение должно иметь подпись внизу (например, «Официальная обложка альбома "Звук Тишины"»).
        *   Вся конструкция (картинка + подпись) должна быть обернута в семантический контейнер для медиа.
        *   Не забудьте про альтернативный текст.
    *   **Аудио-плеер:**
        *   Добавьте возможность прослушать заглавный трек.
        *   Плеер должен отображать стандартные элементы управления (play/pause и т.д.).
        *   Трек должен воспроизводиться по кругу (зациклен).
        *   *Дополнительно:* Укажите несколько источников аудиофайла разного формата (например, mp3 и ogg) для совместимости, используя внутренний тег источника.
    *   **Раздел «Информация об альбоме» (Список определений):**
        *   Заголовок раздела.
        *   Представьте данные в виде пар «Термин — Значение»:
            *   Жанр: Альтернативный рок
            *   Год выпуска: 2026
            *   Длительность: 45 минут
            *   Продюсер: Имя Фамилия
    *   **Раздел «Список композиций»:**
        *   Заголовок раздела.
        *   Нумерованный список из 5-7 треков.
        *   Один из треков пометьте как «Хит сезона», выделив это словосочетание акцентным тегом.
    *   **Видео-клип:**
        *   Вставьте видео с премьерой клипа.
        *   Видео должно иметь постер (изображение-заглушку, которое показывается до нажатия Play).
        *   Видео должно иметь атрибуты для автоматического воспроизведения без звука (так как автоплей со звуком часто блокируется браузерами) и элементы управления.
        *   *Дополнительно:* Укажите несколько источников видео разного качества или формата через внутренние теги источников.
        *   Под видео должна быть подпись (например, «Премьера клипа "Горизонт"»).
        *   Вся конструкция (видео + подпись) должна быть обернута в семантический контейнер.
    *   **Цитата критика:**
        *   Отзыв известного музыкального обозревателя об альбоме, оформленный как блочная цитата.

3.  **Боковая панель (Aside):**
    *   Раздел «Об артисте».
    *   Фотография артиста (обычное изображение, не адаптивное через picture, но с подписью).
    *   Подпись под фото: «Имя Артиста на съемках».
    *   Краткая биография (абзац текста).
    *   Контактные данные менеджера (email) в специальном теге для адресов.

4.  **Подвал сайта:**
    *   Копирайт (год и название лейбла).
    *   Адрес студии звукозаписи (город, улица) в специальном теге.

---

**На что обратить внимание при выполнении (Контроль знаний):**

1.  **`dl`, `dt`, `dd`:** Используйте их строго для раздела «Информация об альбоме», где есть пары ключ-значение.
2.  **`figure` и `figcaption`:**
    *   Для обложки альбома: `<figure>` оборачивает `<picture>` и `<figcaption>`.
    *   Для видео: `<figure>` оборачивает `<video>` и `<figcaption>`.
    *   Для фото артиста в сайдбаре: `<figure>` оборачивает `<img>` и `<figcaption>`.
    *   *Помните:* `figcaption` всегда внутри `figure`.
3.  **`picture` и `source`:** Используйте только там, где нужна адаптивность картинки (обложка альбома). Внутри `picture` должен быть `img` как fallback.
4.  **Медиа-теги:**
    *   `audio` и `video` должны иметь атрибут `controls`, иначе пользователь не сможет ими управлять.
    *   Для `video` с `autoplay` обязательно наличие `muted`.
    *   Внутренние теги `<source>` используются внутри `audio`, `video` и `picture` для указания разных форматов/источников.
5.  **Списки:**
    *   Меню — `ul`/`li`.
    *   Треки — `ol`/`li`.
    *   Биография/отзывы — `p`.
