### Задание 3: «Система Генерации Отчетов с Плоским Структурированием»

**Контекст:**
У вас есть данные о отделах компании, где каждый отдел содержит массив сотрудников. Данные имеют вложенную структуру (массив массивов или массив объектов с вложенными массивами). Ваша задача — «сплющить» эти данные, обработать их и сгруппировать заново.

**Входные данные:**
Массив отделов. Каждый отдел — объект со свойствами `deptName` (строка) и `employees` (массив объектов сотрудников с полями `name` и `salary`).
```javascript
const companyData = [
    {
        deptName: "IT",
        employees: [
            { name: "Alex", salary: 2000 },
            { name: "Maria", salary: 2500 }
        ]
    },
    {
        deptName: "HR",
        employees: [
            { name: "John", salary: 1500 },
            { name: "Elena", salary: 1600 }
        ]
    }
];
```

**Требования к логике (пошагово):**
1.  **Извлечение вложенных массивов:**
    *   Создайте единый плоский массив всех сотрудников из всех отделов.
    *   Используйте метод `flatMap`. Он должен сначала преобразовать каждый отдел в массив его сотрудников (`map`), а затем объединить все эти массивы в один (`flat`).
    *   В результате должен получиться простой массив объектов: `[{name: "Alex", ...}, {name: "Maria", ...}, ...]`.
2.  **Повышение зарплаты (Мутация vs Немутация):**
    *   Создайте **новый** массив сотрудников, где каждому сотруднику из IT-отдела (вам нужно как-то понять, кто из IT, возможно, потребуется добавить поле `dept` в сотрудника на этапе `flatMap` или фильтровать иначе) повысили зарплату на 10%.
    *   *Уточнение для простоты:* Давайте считать, что на этапе `flatMap` вы добавляете каждому сотруднику поле `department`, копируя его из родителя.
    *   Затем используйте `map`: если сотрудник из "IT", его новая зарплата = `oldSalary * 1.1`, иначе — старая зарплата.
    *   Исходный массив сотрудников изменять нельзя.
3.  **Группировка по первой букве имени:**
    *   Возьмите массив сотрудников с обновленными зарплатами.
    *   Используйте метод `reduce`, чтобы сгруппировать их в объект, где ключом является первая буква имени, а значением — массив сотрудников с такой буквой.
    *   Пример результата: `{ "A": [{...}], "M": [{...}], "J": [{...}], "E": [...] }`.
4.  **Сортировка внутри групп:**
    *   Пройдитесь по полученному объекту (через `Object.keys()` или `for...in`) и отсортируйте массивы сотрудников внутри каждой группы по убыванию зарплаты.
    *   Используйте метод `sort` с функцией сравнения чисел.
5.  **Вывод итогов:**
    *   Выведите полученный объект группировки в консоль.
    *   Отдельно выведите общую сумму зарплат всех сотрудников (используя `reduce` на плоском массиве из шага 2).

**Ожидаемый результат:**
Объект, где сотрудники сгруппированы по первой букве имени, внутри групп отсортированы по зарплате (убывание), и общая сумма зарплат по компании.

*Цель: Отработка `flatMap`, сложного `reduce` для группировки, комбинирования методов, работы с вложенными структурами.*
