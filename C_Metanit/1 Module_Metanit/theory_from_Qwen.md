### 1. Структура программы на Си

Любая программа на C состоит из функций. Точкой входа всегда является функция `main`. Без неё программа не запустится.

**Базовая структура:**
```c
#include <stdio.h> // Директива препроцессора: подключает библиотеку стандартного ввода-вывода

// Глобальные переменные (объявляются здесь, если нужны во всей программе)

int main() {
    // Локальные переменные
    // Тело программы
    // Инструкции выполняются последовательно
    
    return 0; // Возврат кода завершения (0 означает успешное выполнение)
}
```

*   `#include <stdio.h>`: Необходим для использования `printf` и `scanf`.
*   `int main()`: Главная функция. `int` означает, что она возвращает целое число.
*   `{ ... }`: Фигурные скобки ограничивают блоки кода.
*   `;`: Каждая инструкция должна заканчиваться точкой с запятой.

### 2. Переменные и Типы данных

Переменная — это именованная область памяти для хранения данных. В C тип переменной определяется строго при объявлении и не может меняться динамически (статическая типизация).

#### Основные типы данных:

| Тип | Размер (байт)* | Диапазон значений | Описание |
| :--- | :--- | :--- | :--- |
| `char` | 1 | -128 до 127 | Символ или маленькое целое |
| `short` | 2 | -32,768 до 32,767 | Короткое целое |
| `int` | 4 | -2 млрд до 2 млрд | Целое число (стандарт) |
| `long` | 4 или 8 | Зависит от системы | Длинное целое |
| `float` | 4 | ~7 знаков после запятой | Число с плавающей точкой (одинарная точность) |
| `double` | 8 | ~15 знаков после запятой | Число с плавающей точкой (двойная точность) |

*\*Размеры могут зависеть от архитектуры компилятора и ОС.*

**Пример объявления:**
```c
int age = 25;
double price = 99.99;
char grade = 'A'; // Одинарные кавычки для символов
```

### 3. Консольный вывод: Функция `printf`

Функция `printf` (print formatted) выводит текст на экран. Она использует **спецификаторы формата** для вставки переменных в строку.

**Синтаксис:**
`printf("Строка с спецификаторами", аргумент1, аргумент2, ...);`

**Основные спецификаторы:**
*   `%d` или `%i` — целое число (`int`)
*   `%f` — вещественное число (`float`, `double`)
*   `%c` — символ (`char`)
*   `%s` — строка (`string`, массив char)
*   `%x` — шестнадцатеричное число
*   `\n` — переход на новую строку
*   `\t` — табуляция

**Пример:**
```c
#include <stdio.h>

int main() {
    int x = 10;
    float y = 3.14;
    char z = 'B';
    
    printf("Целое: %d\n", x);
    printf("Вещественное: %.2f\n", y); // .2 означает 2 знака после запятой
    printf("Символ: %c\n", z);
    printf("Комбинированный вывод: X=%d, Y=%.2f\n", x, y);
    
    return 0;
}
```

### 4. Константы

Константы — это значения, которые нельзя изменить после инициализации.

**Способ 1: Ключевое слово `const`**
```c
const double PI = 3.14159;
// PI = 3.14; // Ошибка компиляции!
```

**Способ 2: Директива препроцессора `#define`**
```c
#define MAX_SIZE 100
// Препроцессор просто заменяет все вхождения MAX_SIZE на 100 до компиляции.
// Не имеет типа данных, используется чаще для простых замен.
```

### 5. Арифметические операции

Стандартные математические действия:

*   `+` Сложение
*   `-` Вычитание
*   `*` Умножение
*   `/` Деление
*   `%` Остаток от деления (только для целых чисел!)

**Важный нюанс деления:**
Если оба операнда целые, результат будет целым (дробная часть отбрасывается).
```c
int a = 5;
int b = 2;
printf("%d\n", a / b); // Выведет 2, а не 2.5

// Чтобы получить дробь, нужно привести хотя бы один операнд к float/double:
printf("%.2f\n", (float)a / b); // Выведет 2.50
```

**Инкремент и декремент:**
*   `++i` (префиксный): сначала увеличивает, потом использует.
*   `i++` (постфиксный): сначала использует, потом увеличивает.

### 6. Операции присваивания

Помимо простого `=`, существуют сокращенные формы:

| Операция | Эквивалент | Описание |
| :--- | :--- | :--- |
| `a += b` | `a = a + b` | Прибавить и присвоить |
| `a -= b` | `a = a - b` | Вычесть и присвоить |
| `a *= b` | `a = a * b` | Умножить и присвоить |
| `a /= b` | `a = a / b` | Разделить и присвоить |
| `a %= b` | `a = a % b` | Остаток и присвоить |

### 7. Преобразование типов (Type Casting)

C позволяет явно или неявно преобразовывать типы.

**Неявное преобразование:**
Компилятор автоматически расширяет тип меньшего размера до большего, чтобы избежать потери данных.
```c
int i = 10;
double d = i; // i автоматически превращается в 10.0
```

**Явное преобразование (Cast):**
Вы принудительно меняете тип.
```c
double pi = 3.14;
int whole = (int)pi; // whole станет 3 (дробная часть отбрасывается)
```

### 8. Условные операции (Операторы сравнения и логики)

Используются для проверки условий. Результат: `1` (истина) или `0` (ложь).

**Сравнение:**
*   `==` Равно (не путать с `=`!)
*   `!=` Не равно
*   `>` Больше
*   `<` Меньше
*   `>=` Больше или равно
*   `<=` Меньше или равно

**Логические:**
*   `&&` И (AND) — истина, если оба условия истинны.
*   `||` ИЛИ (OR) — истина, если хотя бы одно условие истинно.
*   `!` НЕ (NOT) — инверсия (истина становится ложью).

**Пример:**
```c
int age = 20;
int hasMoney = 1;

if (age >= 18 && hasMoney) {
    printf("Доступ разрешен\n");
}
```

### 9. Поразрядные (Битовые) операции

Работают с отдельными битами целых чисел. Используются в системном программировании, работе с железом, оптимизации.

*   `&` (И): Бит равен 1, только если оба бита 1.
*   `|` (ИЛИ): Бит равен 1, если хотя бы один бит 1.
*   `^` (Исключающее ИЛИ/XOR): Бит равен 1, если биты разные.
*   `~` (НЕ): Инвертирует все биты.
*   `<<` Сдвиг влево (умножение на 2 в степени n).
*   `>>` Сдвиг вправо (деление на 2 в степени n).

**Пример:**
```c
unsigned char a = 5;  // 0000 0101
unsigned char b = 3;  // 0000 0011

printf("%d\n", a & b); // 0000 0001 -> 1
printf("%d\n", a | b); // 0000 0111 -> 7
printf("%d\n", a << 1); // 0000 1010 -> 10 (5 * 2)
```

### 10. Условные конструкции

#### `if-else`
```c
int score = 85;

if (score >= 90) {
    printf("Отлично\n");
} else if (score >= 70) {
    printf("Хорошо\n");
} else {
    printf("Плохо\n");
}
```

#### Тернарный оператор
Короткая запись `if-else` для присваивания.
`условие ? значение_если_истина : значение_если_ложь;`

```c
int age = 20;
char* status = (age >= 18) ? "Взрослый" : "Ребенок";
printf("%s\n", status);
```

#### `switch-case`
Удобно для множественного выбора по одному значению.
```c
int day = 3;

switch (day) {
    case 1:
        printf("Понедельник\n");
        break; // Обязательно! Иначе выполнится следующий case
    case 2:
        printf("Вторник\n");
        break;
    case 3:
        printf("Среда\n");
        break;
    default:
        printf("Другой день\n");
        break;
}
```

### 11. Циклы

#### `for`
Когда известно количество повторений.
```c
// for (инициализация; условие; шаг)
for (int i = 0; i < 5; i++) {
    printf("%d ", i); // 0 1 2 3 4
}
```

#### `while`
Выполняется, пока условие истинно. Проверка перед выполнением.
```c
int i = 0;
while (i < 5) {
    printf("%d ", i);
    i++;
}
```

#### `do-while`
Выполняется минимум один раз. Проверка после выполнения.
```c
int i = 0;
do {
    printf("%d ", i);
    i++;
} while (i < 5);
```

**Управление циклом:**
*   `break;` — немедленно прерывает цикл.
*   `continue;` — пропускает остаток текущей итерации и переходит к следующей.

### 12. Введение в массивы и строки

#### Массивы
Набор элементов одного типа, расположенных в памяти подряд. Индексация начинается с **0**.

```c
int numbers[5] = {10, 20, 30, 40, 50};

printf("%d\n", numbers[0]); // 10
printf("%d\n", numbers[4]); // 50

// Изменение элемента
numbers[2] = 99;

// Перебор массива
for (int i = 0; i < 5; i++) {
    printf("%d ", numbers[i]);
}
```
*Важно: C не проверяет выход за границы массива. Если обратиться к `numbers[10]`, программа может упасть или повредить данные.*

#### Строки
В C нет отдельного типа "строка". Строка — это массив символов `char`, заканчивающийся нулевым символом `'\0'`.

```c
char name[] = "Hello"; 
// В памяти: 'H', 'e', 'l', 'l', 'o', '\0'
// Размер массива автоматически станет 6 байт.

char manual[6] = {'H', 'i', '\0'}; // Явное создание строки

printf("%s\n", name); // %s выводит строку до встречи с '\0'
```

Для работы со строками (копирование, сравнение, длина) нужна библиотека `<string.h>`.
*   `strlen(str)` — длина строки (без учета `\0`).
*   `strcpy(dest, src)` — копирование строки.
*   `strcmp(s1, s2)` — сравнение строк (возвращает 0, если равны).

### 13. Ввод в консоли: Функция `scanf`

`scanf` считывает данные из стандартного ввода (клавиатуры) согласно формату.

**Синтаксис:**
`scanf("формат", &переменная1, &переменная2, ...);`

**Критически важно:** Перед именем переменной должен стоять амперсанд `&` (оператор взятия адреса), кроме случаев со строками (массивами).

**Спецификаторы такие же, как у `printf`:**
*   `%d` — для `int`
*   `%f` — для `float`
*   `%lf` — для `double` (важно: для scanf двойная точность требует `lf`, для printf можно `f`)
*   `%c` — для `char`
*   `%s` — для строки (без пробелов!)

**Пример:**
```c
#include <stdio.h>

int main() {
    int age;
    float height;
    char name[50];

    printf("Введите имя: ");
    scanf("%s", name); // Для массивов & не нужен, так как имя массива уже адрес

    printf("Введите возраст: ");
    scanf("%d", &age); // & обязателен

    printf("Введите рост: ");
    scanf("%f", &height);

    printf("Привет, %s! Тебе %d лет, твой рост %.2f м.\n", name, age, height);

    return 0;
}
```

**Проблемы `scanf`:**
1.  **Остановка на пробеле:** `%s` считывает слово до первого пробела, табуляции или новой строки. Если ввести "Ivan Ivanov", в `name` попадет только "Ivan".
    *   *Решение:* Использовать `fgets(name, sizeof(name), stdin);` для ввода строк с пробелами.
2.  **Очистка буфера:** После ввода числа через `scanf` в буфере остается символ новой строки `\n`. Следующий `scanf("%c", ...)` или `fgets` может считать этот остаток вместо нового ввода.
    *   *Решение:* Иногда добавляют пробел в формате `scanf(" %c", &ch);` или используют `getchar()` для очистки.

### Итоговый комплексный пример

```c
#include <stdio.h>
#include <string.h>

#define MAX_NAME_LEN 50

int main() {
    // 1. Объявление переменных
    char name[MAX_NAME_LEN];
    int age;
    double salary;
    const double TAX_RATE = 0.13; // Константа

    // 2. Ввод данных
    printf("=== Анкета сотрудника ===\n");
    
    printf("Введите имя: ");
    // Используем fgets для безопасного ввода строки с пробелами
    if (fgets(name, MAX_NAME_LEN, stdin) != NULL) {
        // Удаляем символ новой строки, который добавляет fgets
        size_t len = strlen(name);
        if (len > 0 && name[len - 1] == '\n') {
            name[len - 1] = '\0';
        }
    }

    printf("Введите возраст: ");
    scanf("%d", &age);

    printf("Введите зарплату: ");
    scanf("%lf", &salary);

    // 3. Обработка данных
    double tax = salary * TAX_RATE;
    double netSalary = salary - tax;

    // 4. Условная конструкция
    char* status;
    if (age >= 60) {
        status = "Предпенсионер";
    } else if (age >= 18) {
        status = "Совершеннолетний";
    } else {
        status = "Несовершеннолетний";
    }

    // 5. Вывод результатов
    printf("\n--- Результат ---\n");
    printf("Имя: %s\n", name);
    printf("Возраст: %d лет (%s)\n", age, status);
    printf("Зарплата gross: %.2f руб.\n", salary);
    printf("Налог (%.0f%%): %.2f руб.\n", TAX_RATE * 100, tax);
    printf("Зарплата net: %.2f руб.\n", netSalary);

    // 6. Поразрядная операция (пример)
    int flags = 5; // 101 в двоичной
    int mask = 1;  // 001 в двоичной
    if (flags & mask) {
        printf("Первый бит установлен.\n");
    }

    return 0;
}
```
