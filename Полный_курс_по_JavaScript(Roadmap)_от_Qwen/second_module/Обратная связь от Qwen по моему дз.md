## 🔹 Задание 2: `getUniqueWords(text)` — уникальные слова

### Ваш код:
```js
function getUniqueWords(text) {
    const array = text.toLowerCase()
        .replace(/[^\w\sа-яё]/gi, ' ')
        .replace(/\s+/g, ' ')
        .trim()
        .split(" ");
    let lst = [];
    for (let i = 0; i < array.length; i++) {
        if (!lst.includes(array[i])) {
            lst.push(array[i]);
        }
    }
    return lst;
}
```

### Анализ:
🔥 **Это лучшее выполнение из всех трёх заданий!**

✅ Вы **обработали строку как профессионал**:
- Привели к нижнему регистру → `"Привет"` и `"привет"` считаются одинаково.
- Удалили **все не-буквы и не-пробелы** (включая русские буквы `а-яё`) — регулярка грамотная.
- Заменили множественные пробелы на один → избегаете пустых строк.
- Убрали пробелы по краям → `trim()`.

✅ Алгоритм поиска уникальных слов через `includes` — **простой и понятный** (хотя не самый быстрый для больших данных, но для учебы — отлично).

✅ Тестовый пример работает:
```js
"Привет№ как, дела! как привет дела"
→ ["привет", "как", "дела"]
```

⚠️ **Мелкие замечания**:
1. Переменная `lst` — лучше назвать `uniqueWords` (читаемость).
2. Можно было использовать `Set` для ещё более краткого решения:
   ```js
   return [...new Set(array)];
   ```
   Но ваш способ **полностью соответствует уровню модуля** — вы ещё не проходили `Set`, так что это **не ошибка**, а просто альтернатива.

🟢 **Вердикт**: **отлично**. Полное соответствие требованиям + продуманная обработка текста.
